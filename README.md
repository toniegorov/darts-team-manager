# Дартс · Командный менеджер

Десктопное приложение для ведения командных матчей по дартс. Доступно в двух вариантах интерфейса:

- **CustomTkinter** — оригинальная версия (`darts_team_manager.py`)
- **PySide6 (Qt6)** — новая версия (`darts_team_manager_pyside6.py`)

Обе версии полностью идентичны по логике и функциональности, используют одну и ту же базу данных `darts_manager.db`.

## Возможности

**Управление участниками и командами**

- Добавление, переименование и удаление участников
- Формирование команд (от 2 до 5) с настраиваемыми названиями и цветами
- Сохранение составов между сессиями

**Матч — 9 основных игр**

Матч состоит из 9 игр, расположенных в сетке 3 строки по 3 колонки:

| Строка | Тип игры | Правило |
|--------|----------|---------|
| 1 | 3 игры на **максимум** | Побеждает команда с большей суммой |
| 2 | 3 игры на **минимум** | Побеждает команда с меньшей суммой |
| 3 | 3 игры на **бул** | Значения от −25 до 150, побеждает большая сумма |

При ничьей в раунде автоматически появляется **доп. игра (по 1 дротику)** — справа от основной игры.

**Ввод результатов**

- Стандартный ввод — одно число (общий результат игрока)
- Поддержка арифметических выражений (например, `20+5` автоматически вычисляется в `25`)
- Ввод **«x»** — пропуск броска (не учитывается в сумме команды и статистике игрока)
- Отрицательные значения разрешены только в играх на бул
- Навигация между полями клавишами **Enter**, **Tab**, **стрелки вверх/вниз**

**Сектора — 2 дополнительные игры**

После 9 основных игр доступна галочка **«Сектора»**. При включении появляются 2 одинаковые таблички:

- Соперник выбирает 3 сектора
- Каждый участник обеих команд бросает по 1 дротику в каждый сектор
- Количество попаданий записывается в поле рядом с именем
- Команда с большим числом попаданий побеждает в секторах
- Каждая победа в секторах даёт **+1 к общему счёту** матча

**Сохранение и статистика**

- Ручное сохранение матча (кнопка «Сохранить матч» активна после определения победителя)
- Просмотр сохранённых матчей (двойной клик по записи в истории)
- Множественное удаление матчей из истории (Ctrl+клик для выбора нескольких)
- Фильтрация истории по дате
- Статистика игроков: средние по макс/мин/бул, ранги, общее место
- Графики динамики результатов (двойной клик по игроку в статистике)

**Интерфейс**

- Тёмная тема (Catppuccin Mocha)
- Компактное табло счёта
- Скроллируемая область матча

## Требования

- **Python 3.9+**

Для CustomTkinter версии:
- **customtkinter** — `pip install customtkinter`
- **tkinter** (обычно входит в стандартную поставку Python)

Для PySide6 версии:
- **PySide6** — `pip install PySide6`
- **matplotlib** — `pip install matplotlib` (для графиков)

## Установка

**1. Клонировать репозиторий**

```bash
git clone https://github.com/toniegorov/darts-team-manager.git
cd darts-team-manager
```

**2. Установить зависимости**

```bash
pip install -r requirements.txt
```

На Linux, если tkinter не установлен (для CustomTkinter версии):

```bash
# Ubuntu / Debian
sudo apt-get install python3-tk

# Fedora
sudo dnf install python3-tkinter

# Arch
sudo pacman -S tk
```

## Запуск

**CustomTkinter версия (оригинал):**

```bash
python darts_team_manager.py
```

**PySide6 версия (Qt6):**

```bash
python darts_team_manager_pyside6.py
```

При первом запуске автоматически создаётся файл базы данных `darts_manager.db` в текущей директории. Все участники, составы команд и история матчей сохраняются в нём.
